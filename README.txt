ミュージックエディターの操作説明

マウスは使わない

EscapeキーでXMLデータの編集画面とノーツ編集画面の遷移を行う

○移動
↑↓キーで拍の移動
←→キーで分数の変化(4,8,12,16,24,32,48)※←キーで減少、→キーで増加
小節線以外で←→キーを押すと小節線に戻す

○入力キー
A  S  D  F  G  H
  Z  X  C  V  B  N
Aが一番左、Nが一番右のレーンを指している

これ単体で押すと単ノーツが生成される

○ロングノーツの生成
左Shift+入力キー → 左Shiftを放す

左Shiftを押しながら入力キーを押す
その後伸ばしたい位置まで↑キーで伸ばす
左Shiftを放すとその位置まで生成される


○スライドの生成
右Shift+入力キー → 右Shift+入力キー

右Shiftを押しながら入力キーを押す。この入力を始点とする
次に伸ばしたい位置まで↑キーで伸ばす
その後右Shiftを押しながら入力キーを押す。この入力を終点とする
すると始点と終点が結ばれる


○フリックの生成
Spaceキー+入力キー → ","キー or "."キー
