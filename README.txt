ミュージックエディターの操作説明

マウスは使わない

EscapeキーでXMLデータの編集画面とノーツ編集画面の遷移を行う

○移動
↑↓キーで拍の移動
←→キーで分数の変化(4,8,12,16,24,32,48)※←キーで減少、→キーで増加
小節線以外で←→キーを押すと小節線に戻す

○入力キー
A・Z・S・X・D・C・F・V・G・B・H・N
Aが一番左、Nが一番右のレーンを指している

これ単体で押すと単ノーツが生成される

○左のShift+入力キー
ロングノーツの生成

最初に左Shiftを押しながら入力キーを押す
その後伸ばしたい位置まで↑キーで伸ばす
左Shiftを放すとその位置まで生成される
